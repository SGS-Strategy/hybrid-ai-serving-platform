# Private Cloud Handoff

이 디렉터리는 Private Cloud Foundation에서 다른 역할로 넘길 공개 가능한 산출물과 계획을 정리합니다.

## 인계 대상

| 항목 | 전달 대상 |
| --- | --- |
| OpenStack VM role inventory 기준 | Kubernetes/bootstrap 담당 |
| Kubernetes namespace/RBAC 기준 | Model, worker, monitoring 담당 |
| StorageClass/PVC 기준 | Model build, artifact 담당 |
| GitLab/GPU SSH runner 기준 | Model training 담당 |
| Harbor VM / Argo / Kaniko 계획 | Model packaging 담당 |
| Reverse proxy/DNS 계획 | 운영 담당 |
